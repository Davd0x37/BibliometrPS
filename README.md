# BibliometrPS

## Deployment
First install [Node.js](https://nodejs.org/en/). Then clone repo and run `npm install`.
To install yarn use `npm install -g yarn`

To build scss
```
npm run build
or
yarn build
```

To clean compiled styles
```
npm run clean
or
yarn clean
```

To compile styles
```
npm run styles
or
yarn styles
```
